## Описание

Набор консольных игр для тренировки мозга.


## Игры

### brain-even
Игра "Проверка на чётность". Нужно определить, является ли число чётным.



### brain-calc
Игра "Калькулятор". Нужно вычислить результат математического выражения.



### brain-gcd
Игра "НОД". Нужно найти наибольший общий делитель двух чисел.
